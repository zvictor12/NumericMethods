# Numeric Methods

## О проекте

C# проект с реализациями численных методов для интегрирования и решения систем ОДУ. Результаты расчетов сохраняются в виде графиков в папки `plots` и `intplots`.

## Что внутри

- `Task1.cs` — квадратурные формулы Ньютона-Котса и Гаусса для интеграла с весовой особенностью, составные формулы и оптимальный шаг;
- `Task2.cs` — методы Рунге-Кутты второго и четвертого порядка для системы ОДУ, контроль погрешности и графики сходимости;
- `Task3.cs` — сравнение методов средних прямоугольников, трапеций, Симпсона, Ньютона-Котса и Гаусса с оценкой погрешности;
- `Program.cs` — точка запуска расчетов.

## Запуск

```bash
dotnet run
```
